# Pr_Kredit_Motor_Fortuna (Fortuna Motor Management)

Aplikasi manajemen sistem kredit sepeda motor untuk dealer "Fortuna Motor". Proyek ini mengintegrasikan mobile frontend (Android Java) dengan backend API (PHP & MySQL).

## 🚀 Fitur Utama
- **Manajemen Data Master**: CRUD lengkap untuk data Motor, Kreditor, dan Petugas.
- **Sistem Pengajuan Kredit**: Kalkulasi otomatis harga tunai, uang muka (DP), bunga, tenor, dan angsuran per bulan.
- **Pembayaran Angsuran**: 
  - Deteksi otomatis nomor angsuran berikutnya (*Auto-increment sequence*).
  - Validasi status lunas atau sisa angsuran.
- **Standardisasi Rupiah**: Implementasi format mata uang Rupiah (`Rp 1.000.000`) di seluruh aplikasi.
- **Cetak Invoice PDF**: Menghasilkan bukti pembayaran digital dalam format PDF yang rapi.

## 🛠️ Tech Stack
- **Frontend**: Java (Android Studio)
- **Backend**: PHP (REST API)
- **Database**: MySQL (XAMPP)
